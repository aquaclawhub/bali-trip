DAY 3 · 4/27（周一）

Sanur 海滩 + Beach Club（轻松日）
休闲
📋 今日概要：上午 Sanur 海滩玩沙踏浪 → 海滩 café 午餐冲淡换衣 → 下午 Villa 休息 SPA → 傍晚 Beach Club 看日落。Waterbom 第二天，轻松节奏。
时间安排备注
08:00 出发前往 Sanur Bluebird/Grab，约30分钟车程
08:30 抵达 Sanur 海滩（Mertasari Beach） 公共海滩，免费入场
08:30–12:00 Sanur 海滩玩沙踏浪 东海岸浅滩，无浪，3岁可下海；带沙滩垫/泳衣
12:00 海滩 café 午餐 + 冲淡换衣 点餐后用 café 设施淡水冲脚换衣服
13:30 返回 Seminyak 约30分钟车程，孩子可在车上睡
14:00–16:00 Villa 午休 / 大人 SPA Seminyak 按摩/SPA 选择极多，可提前预约
16:30 出发前往 La Plancha 步行可达，位置在 Seminyak 海滩
17:00 La Plancha 看日落 彩色懒人沙发，低消约 IDR 150,000–300,000/人
19:00 Seminyak 区域晚餐 推荐餐厅：Potato Head 系列 / Revolver / Biku
🚗 Seminyak → Sanur 交通（点击查看）
推荐方式：Bluebird 出租车 / Grab / GoCar
  • 车程：约 25–35 分钟（取决于路况）
  • 费用：约 IDR 80,000–150,000（≈ ¥40–75）
  • 叫车：从 Seminyak 任意地点叫车，直接输入 "Sanur Beach" 导航即可
  • 回程：同方式返回，或让包车司机等候一天（可和 Day 3 司机协商）
💡 小提示：如果 Day 4 的包车司机服务满意，可以直接问司机是否愿意 Day 3 也跟着，包一天车价格比单次叫车更划算，且有空调车随时等候，孩子玩累了可以随时上车休息。
🏖️ Sanur 海滩玩水攻略（点击查看）
推荐海滩：Mertasari Beach
Sanur 最适合亲子下海的海滩，东海岸有珊瑚礁保护，海水平静、浪小，3岁孩子可以安全踏浪。
公共设施
  • 公共淋浴室：有冷水冲洗，设施较基础（无热水），通常免费或收少量费用
  • 更衣区：较简陋，建议在海滩 café 里换好衣服过来
  • 最推荐：找一家海滩 café 坐下，点杯饮料/吃个午饭，用他们的卫生间和冲洗设施，比公共淋浴室舒服很多
建议自带
大沙滩巾 / 浴巾、泳衣 + 备用换洗衣物、沙滩玩具（如想玩沙）、防晒霜、帽子
🏖️ 海滩俱乐部攻略 La Plancha（点击查看）
  • 入场：免费，但必须消费
  • 低消：约 IDR 150,000–300,000/人（≈ ¥75–150/人）
  • 特色：彩色懒人沙发 + 直接沙滩 + 著名日落景观，超适合拍照
  • 儿童：可以带娃，主要是大人的放松环境
  • 时间：建议 16:30 前到，傍晚是黄金时段
  • 地址：Jl. Arjuna, Seminyak（步行可达 Seminyak 各 villa 区域）
